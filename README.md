## How to Use This Repository

### Switching to Your Branch

To work on a specific semester’s projects, switch to the correct branch using Git:
```sh
git checkout s25  # Replace with your assigned branch
```

Or, if you haven’t downloaded the branch yet:
```sh
git fetch --all
git checkout s25
```

### Committing Your Changes
1. Make your edits.
2. Stage and commit your changes:
```sh
git add .
git commit -m "Updated project for s25"
git push origin s25  # Push to your assigned branch
```

If you accidentally commit to the wrong branch, you can undo it using:
```sh
git reset --soft HEAD~1  # Removes the last commit but keeps changes
```
